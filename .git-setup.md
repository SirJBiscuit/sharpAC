# Git Repository Setup

## Repository Information

**GitHub Repository:** https://github.com/SirJBiscuit/sharpAC.git

## Initial Setup (First Time)

### 1. Initialize Git Repository

```bash
cd C:\Users\Jeremiah Payne\CascadeProjects\sharp-ac-remote
git init
git add .
git commit -m "Initial commit - Sharp AC Remote with Phone IR support"
```

### 2. Add Remote Repository

```bash
git remote add origin https://github.com/SirJBiscuit/sharpAC.git
```

### 3. Push to GitHub

```bash
git branch -M main
git push -u origin main
```

## Updating the Repository

### After Making Changes

```bash
git add .
git commit -m "Your commit message here"
git push
```

## Deploying to Server from GitHub

### On Your Server

```bash
# Clone the repository
cd ~/projects
git clone https://github.com/SirJBiscuit/sharpAC.git sharp-ac-remote
cd sharp-ac-remote

# Configure environment
cp .env.example .env
nano .env  # Add your Cloudflare tunnel token

# Deploy
docker-compose up -d
```

## Updating Server from GitHub

```bash
cd ~/projects/sharp-ac-remote
git pull
docker-compose down
docker-compose build
docker-compose up -d
```

## .gitignore

Already configured to ignore:

## Recommended Workflow

1. **Develop locally** on Windows
2. **Test** with `npm run dev`
3. **Commit and push** to GitHub
4. **Pull on server** and redeploy
